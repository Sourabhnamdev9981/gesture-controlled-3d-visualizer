# 3D Hand Gesture Control System

A real-time interactive 3D visualization system controlled using hand gestures through the webcam.

## 🚀 Features

- Real-time hand tracking using camera
- Gesture-based control of 3D object
- Dynamic particle-based 3D shapes
- Multiple shape transformations (sphere, cube, galaxy, etc.)
- Interactive control panel for visuals and behavior
- Smooth animations and transitions

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Three.js
- MediaPipe Hands
- Web Camera API (getUserMedia)

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Open `index.html` using Live Server (recommended)

OR

Simply double-click `index.html` to run in browser

## 🎮 Controls

- Move hand → Move 3D object
- Open/Close hand → Scale object
- Rotate hand → Rotate object
- Use control panel to change:
  - Shape
  - Color theme
  - Glow intensity

## ⚠️ Requirements

- Webcam access must be enabled
- Works best on modern browsers (Chrome recommended)
